Acerca de nuestro grupo

Somos un grupo de estudiantes de el Instituto Tecnológico Superior que realiza el segundo año del curso de informática. Como proyecto de fin de año debemos crear los planos de una arena, diseñar este sitio web y desarrollar un software.

Acerca de nuestro software

El software que debemos crear tiene como objetivo vender entradas a los clientes para que puedan asisitir a los eventos que promocionamos aquí. Este programa esta desarrollado en lenguaje Java y su base de datos hecha en SQL server
